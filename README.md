# 觉刻 Qbit

> 记录即观测，观测带来改变。

觉刻（Qbit）是一款帮助你觉察时间去向的 Android 应用。它会在你设定的间隔（默认每小时）以闹钟的方式打断你的"自动驾驶"，让你停下来回答一个简单的问题：**过去这段时间，你做了什么？**

## 名称含义

**觉刻**——觉察每一刻。"觉"是觉醒、觉察，"刻"是时刻，也是刻录。每一次记录都是一次对自己的觉察。

**Qbit**——取自量子比特（Qubit）。量子态在被观测前处于叠加态，观测的瞬间坍缩为确定态。你的生活也是如此：不记录时浑浑噩噩是"叠加态"，记录的那一刻就"坍缩"成了确定的改变。Qbit 也暗合"bit by bit"——一点一滴，积少成多。

## 灵感来源

灵感来自知乎上的一篇回答。作者提出了一个深刻的观点：

> *记录自己，本质上是对肉身负责，夺过大脑的自动驾驶，真正把自己放到"观测者"的位置上，去过好自己的这一生。*

大多数人每天看起来忙忙碌碌，但注意力带宽往往被无意义地消耗。如果不去记录，你这辈子可能只有一天的生命体验，然后重复到老。开启记录，就是用元认知捕捉潜意识，让冰川下的宝藏一点点浮出水面。

记录 → 发现 → 改变 → 正向反馈 → 掌控感 → 满足感。这是普通人成本最小的自我正向激励闭环。

觉刻就是为了降低这个闭环的门槛而诞生的。你不需要写长篇日记，不需要复杂的时间管理系统，只需要每小时花几秒钟，诚实地记下自己在做什么。日积月累，当你翻看这些记录时，会触目惊心地发现那些被浪费的时间——而这种震动本身，就是改变的开始。

## 功能特性

**闹钟式提醒**：到时间后弹出全屏提醒界面，配合铃声和振动，必须手动关闭。不是一条容易忽略的通知，而是一次真正的"打断"，让你从自动巡航中醒来。

**极简记录**：点击"去记录"后弹出输入框，写下过去这段时间你做了什么，几秒钟就能完成。不强调格式，不要求分类，降低一切记录阻力。

**历史日历**：按月浏览所有记录，有记录的日期会标记圆点，一眼看出哪些天在记录、哪些天中断了。支持自由翻阅过去和未来的月份。

**节假日标注**：日历自动标注法定节假日（红色）、调休补班（蓝色）和普通周末（灰色）。数据来源为国务院办公厅官方发布的放假安排，切换到未缓存的年份时自动获取。

**免打扰规则**：可以设置多条免打扰时段，支持"每天""仅工作日""仅周末"三种模式，工作日和周末的判定会结合节假日数据（调休补班算工作日，法定节假日算休息日）。睡觉时间、会议时间，设好规则就不会被打扰。

**可调间隔**：提醒间隔支持 15 分钟到 3 小时自由选择，找到适合自己节奏的频率。

## 技术栈

Kotlin + Jetpack Compose + Room + Material Design 3，纯本地存储，无需注册，不上传任何数据。节假日数据通过 jsdelivr CDN 获取自 [holiday-cn](https://github.com/NateScarlet/holiday-cn) 开源项目。
